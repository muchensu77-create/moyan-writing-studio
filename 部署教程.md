# 代写工作室个人主页 - 完整部署教程

## 文件说明

```
小红书个人空间/
├── index.html      ← 主页面（所有内容都在这里）
├── style.css       ← 样式文件（控制颜色、布局、动画）
├── script.js       ← 交互脚本（数字动画、滚动效果）
└── 部署教程.md      ← 本教程
```

## 第一步：个性化修改内容

### 1. 修改工作室名称
在 `index.html` 中搜索 **"墨言工作室"**，全部替换为你的工作室名称。

### 2. 修改联系方式
找到 `<!-- 联系我们 -->` 部分，修改以下内容：
- `your_wechat_id` → 你的微信号
- `@墨言工作室` → 你的小红书账号名
- `your_email@example.com` → 你的邮箱

### 3. 替换二维码图片
找到 `<div class="qr-placeholder">微信二维码</div>`，替换为：
```html
<img src="你的二维码图片.jpg" alt="微信二维码" style="width:150px; border-radius:12px;">
```

### 4. 修改价格
在 `<!-- 价格方案 -->` 部分，修改数字和描述。

### 5. 修改数据统计
在 Hero 区域找到 `data-target="3000"` 等，改成你的实际数据。

### 6. 修改客户评价
在 `<!-- 客户评价 -->` 部分，替换为真实的客户反馈。

## 第二步：修改颜色主题（可选）

在 `style.css` 文件顶部的 `:root` 中修改颜色：

```css

    --accent: #e94560;       /* 强调色（红色按钮） */
```

## 第四步：免费部署上线（三种方案）

### 方案一：GitHub Pages（推荐，完全免费）

1. **注册 GitHub 账号**
   - 访问 https://github.com 注册

2. **创建仓库**
   - 点击右上角 "+" → "New repository"
   - 仓库名填：`你的用户名.github.io`（例如 `moyan-studio.github.io`）
   - 选择 Public（公开）
   - 点击 Create repository

3. **上传文件**
   - 在新建的仓库页面点击 "uploading an existing file"
   - 把 `index.html`、`style.css`、`script.js` 和图片文件拖进去
   - 点击 "Commit changes"

4. **开启 Pages**
   - 进入仓库 → Settings → Pages
   - Source 选择 "Deploy from a branch"
   - Branch 选择 "main"，文件夹选 "/(root)"
   - 点击 Save

5. **访问你的网站**
   - 等待1-2分钟
   - 访问 `https://你的用户名.github.io`

### 方案二：Vercel（推荐，自动部署）

1. 访问 https://vercel.com 用 GitHub 登录
2. 点击 "Import Project" → 选择你的仓库
3. 点击 Deploy
4. 自动生成域名，如 `xxx.vercel.app`

### 方案三：Netlify（同样免费）

1. 访问 https://www.netlify.com
2. 注册后直接拖拽文件夹到部署区域
3. 自动生成域名

## 第五步：绑定自定义域名（可选）

如果你想用自己的域名（如 `www.moyan.com`）：

1. **购买域名**
   - 阿里云：https://wanwang.aliyun.com
   - 腾讯云：https://dnspod.cloud.tencent.com
   - 价格：约 30-60 元/年

2. **配置 DNS 解析**
   - 在域名管理后台添加 CNAME 记录
   - GitHub Pages：指向 `你的用户名.github.io`
   - Vercel：指向 `cname.vercel-dns.com`

3. **在平台设置自定义域名**
   - GitHub：仓库 Settings → Pages → Custom domain
   - Vercel：项目 Settings → Domains

## 常见问题

### Q: 修改后网站没变化？
A: 浏览器有缓存，按 `Ctrl + Shift + R` 强制刷新。

### Q: 手机上显示不正常？
A: 网站已做了响应式适配，确保没有改动 CSS 中的 `@media` 部分。

### Q: 图片加载慢？
A: 压缩图片后再上传，推荐用 https://tinypng.com 压缩。

### Q: 想添加更多页面？
A: 复制 `index.html` 改名（如 `about.html`），修改内容，在导航栏添加链接即可。

### Q: 表单提交怎么接收？
A: 当前表单只是前端展示。如需真正接收，可以：
- 使用 Formspree（https://formspree.io）免费表单服务
- 将 form 的 action 改为 Formspree 提供的链接
